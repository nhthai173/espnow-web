# Giới thiệu

Đây là server giả lập lại thiết bị gateway bằng nodejs

# Yêu cầu

### 1. Nodejs

### 2. express
```bash
npm install express
```

# Sử dụng

Chạy file `main.js` để chạy server. Server được chạy ở port 3000
